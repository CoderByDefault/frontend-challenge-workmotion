# Workmotion frontend-challenge

## Using JSON server 
[json-server documentation](https://www.npmjs.com/package/json-server)

`
npm install -g json-server
`

run it on port 3001

`
json-server -p 3001 --watch db.json
`


## Running the app

`
npm install
`

then

`
npm start
`
app will be running on localhost port 3000
